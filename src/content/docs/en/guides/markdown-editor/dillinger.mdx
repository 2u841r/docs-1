[Dillinger](https://dillinger.io/) is an online, in-browser Markdown editor that can link with your GitHub account to edit, push, and pull `.md` and `.mdx` files from a GitHub-hosted repository.

Dillinger is a solution that allows contributors to collaborate on your Astro project's content files.

## Prerequisites

- An existing Astro project.
- A free Dillinger account.
