[StackEdit](https://stackedit.io/) is an online Markdown editor that can link with your GitHub account to edit, push, and pull `.md` and `.mdx` files from a hosted repository.

StackEdit is a solution that allows contributors to collaborate on your Astro project's content files.

## Prerequisites

- An existing Astro project.
- A free StackEdit account.
