[JekyllPad](https://jekyllpad.com/) is an in-browser, WYSIWYG-style Markdown editor for editing content files in your GitHub-hosted repositories.

JekyllPad is a solution that allows contributors to collaborate on your Astro project's content files.

## Prerequisites

- An existing Astro project.
- A free JekyllPad account.
